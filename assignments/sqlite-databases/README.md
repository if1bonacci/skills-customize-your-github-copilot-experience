# 📘 Assignment: Introduction to Databases with SQLite

## 🎯 Objective

Learn how to integrate a SQLite database with Python. By the end of this assignment, you will be able to perform CRUD operations on a database and connect your FastAPI or Python app to persistent storage.

## 📝 Tasks

### 🛠️ Task 1: Set Up SQLite Database

#### Description
Create a new SQLite database and a table for storing resources (e.g., books, tasks, or users).

#### Requirements
Completed program should:
- Use Python's `sqlite3` module to create a database file
- Define a table with appropriate columns (e.g., id, title, description)
- Include a script to initialize the database

### 🛠️ Task 2: CRUD Operations in Python

#### Description
Write Python functions to Create, Read, Update, and Delete records in your SQLite database.

#### Requirements
Completed program should:
- Implement functions for each CRUD operation
- Use parameterized queries to prevent SQL injection
- Print results to verify each operation

### 🛠️ Task 3: (Extension) Integrate with FastAPI

#### Description
(Optional) Connect your FastAPI app to the SQLite database and expose CRUD endpoints.

#### Requirements
Completed program should:
- Use FastAPI endpoints to interact with the database
- Return JSON responses for each operation
- Handle errors gracefully

---

**Starter code is provided in `starter-code.py`.**
